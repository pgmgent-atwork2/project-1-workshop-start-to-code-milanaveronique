How to make this game?

In this tutorial, we will go through the steps to create the "Guess the Movie" game. We will cover the following topics:

  • What do you need?
  • Setting up the file structure
  • Creating the HTML structure
  • Styling the game with CSS
  • Adding interactivity with JavaScript

Step 1. What do you need?

To create this game, you will need the following tools:

  • A code editor (e.g., Visual Studio Code, Sublime Text, Atom)
  • A web browser (e.g., Google Chrome, Mozilla Firefox, Safari)

1.1 Setting up the Development Environment

In our tutorial, we will be using Visual Studio Code as our code editor and Google Chrome as our web browser. Follow the links to download and install these tools through the installation guide if you haven't already. Once you have your code editor and web browser set up, you're ready to start building the game!

1.2 Downloading the Starter Template

To make everything easier, we will provide you with a starter template and detailed instructions for each step of the development process. Download the starter template.

1.3 Exploring the Starter Template

After downloading the starter template, extract the files and open the project in Visual Studio Code. Open Visual Studio Code and select "File" > "Open Folder" to open the extracted folder.

Take a moment to explore the file structure and familiarize yourself with the different files and folders. The structure should look something like this:

1.4 Installing Dependencies

Before we start coding, we need to install the necessary dependencies for our project. Open the terminal in Visual Studio Code by selecting "Terminal" > "New Terminal". In the terminal, run the following command to install the dependencies:

npm install

This command will install all the required packages and libraries that we will be using in our project. Once the installation is complete, you are ready to start coding!

1.5 Running the Development Server

To run the development server, go back to the terminal and run the following command:

npm run dev

This will start the development server and open the game in your default web browser.

4.2: Implementing the game logic

Now that we have our data defined, we can implement the game logic in the main.js file. This will include functions to initialize the game, handle user guesses, update the UI, and manage the game state. Right-click on the scripts folder and select "New File" to create a new JavaScript file. Name it main.js and add the following code: 

import '../styles/main.css';
import { movieList } from './data';
const maxTries = 5;
let triesLeft = maxTries;

This imports our styling and our data from data.js. This initializes the game with a maximum number of tries and sets the current number of tries left.

Next, we will implement the functions to initialize the game. Add the following code: 


const movieEmoji = document.getElementById('movie-emoji');
const movieHint = document.getElementById('movie-hint');

let currentMovie = null;

function initializeGame() {
  if (!movieEmoji || !movieHint) {
    return;
  }
  triesLeft = maxTries;
  movieHint.classList.add('no-display');
  const randomMovie = movieList[Math.floor(Math.random() * movieList.length)];
  if (currentMovie && randomMovie.name === currentMovie.name) {
    return initializeGame();
  }
  currentMovie = randomMovie;
  movieEmoji.textContent = randomMovie.emoji;
  movieHint.textContent = randomMovie.hint;
}

if (movieEmoji && movieHint) {
  initializeGame();
}

This code defines the function to initialize the game. It randomly selects a movie from the list, updates the UI with the movie's emojis and hint, and resets the number of tries left.

Now let's implement the game logic to handle user guesses and update the UI accordingly.


const guessInput = document.getElementById('guess-input');
const submitButton = document.getElementById('submit-guess');
const triesLeftDisplay = document.getElementById('tries-left');
const modal = document.getElementById('modal');
const modalMessage = document.getElementById('modal-message');
if (guessInput && submitButton && triesLeftDisplay && modal && modalMessage) {
  submitButton.addEventListener('click', () => {
    const userGuess = guessInput.value.trim().toLowerCase();
    const correctAnswer = currentMovie.name.trim().toLowerCase();
    const closeAnswer = currentMovie.close ? currentMovie.close.trim().toLowerCase() : null;

    if (userGuess === correctAnswer || userGuess.includes(correctAnswer)) {
      modalMessage.innerHTML = '<img src="../src/assets/correct.gif" alt="Correct guess"> <p>Congratulations! You guessed the movie!</p>';
      modal.classList.remove('no-display');
      setTimeout(() => {
        modal.classList.add('no-display');
      }, 3000);
      resetGame();
    } else if (userGuess === closeAnswer && userGuess.includes(closeAnswer)) {
      triesLeft--;
      triesLeftDisplay.textContent = `Tries left: ${triesLeft}`;
      guessInput.value = '';
      if (triesLeft === 3 && movieHint) {
        movieHint.classList.remove('no-display');
      }
      if (triesLeft === 0) {
        modalMessage.innerHTML = `<img src="../src/assets/game-over.gif" alt="Game over"> <p>Game over! The correct answer was: <span class="highlight">${currentMovie.name}</span></p>`;
        modal.classList.remove('no-display');
        resetGame();
        setTimeout(() => {
          modal.classList.add('no-display');
        }, 3000);
      } else {
        modalMessage.innerHTML = '<img src="../src/assets/close.gif" alt="Close guess"> <p>Close! You are on the right track. Try again.</p>';
        modal.classList.remove('no-display');
        setTimeout(() => {
          modal.classList.add('no-display');
        }, 3000);
      }
    }
    else {
      triesLeft--;
      triesLeftDisplay.textContent = `Tries left: ${triesLeft}`;
      guessInput.value = '';
      if (triesLeft === 3 && movieHint) {
        movieHint.classList.remove('no-display');
      }
      if (triesLeft === 0) {
        modalMessage.innerHTML = `<img src="../src/assets/game-over.gif" alt="Game over"> <p>Game over! The correct answer was: <span class="highlight">${currentMovie.name}</span></p>`;
        modal.classList.remove('no-display');
        resetGame();
        setTimeout(() => {
          modal.classList.add('no-display');
        }, 3000);
      } else {
        modalMessage.innerHTML = '<img src="../src/assets/wrong.gif" alt="Wrong guess"> <p>Wrong guess! Try again.</p>';
        modal.classList.remove('no-display');
        setTimeout(() => {
          modal.classList.add('no-display');
        }, 3000);
      }
    }
  });
}

This code adds an event listener to the submit button that checks the user's guess against the correct answer. It also checks if the guess is close to the correct answer (if a close answer is defined) and provides feedback accordingly. The code updates the number of tries left, shows hints, and displays modals for correct and incorrect guesses.

Now we will add code to reset the game. Add the following code:


  function resetGame() {
  triesLeft = maxTries;
  triesLeftDisplay.textContent = `Tries left: ${triesLeft}`;
  guessInput.value = '';
  movieHint.classList.add('no-display');

  initializeGame();

}

We also add a small snippet of code to close the popup. Add the following code:

const closeModalButton = document.getElementById('close-modal');
if (closeModalButton && modal) {
  closeModalButton.addEventListener('click', () => {
    modal.classList.add('no-display');
  });
}

Finally, we will implement the function to reset the game when the user clicks the "Next Movie" button. Add the following code: 

const resetButton = document.getElementById('new-game');
if (resetButton) {
  resetButton.addEventListener('click', resetGame);
}

This code adds an event listener to the "Next Movie" button that calls the resetGame function to start a new game.
